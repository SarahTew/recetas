About

Recetas de las Américas is a web project from The US Caribbean & Ethnic Florida Digital Newspaper Project using newspapers digitized as part of the National Digital Newspaper Program(NDNP), a federal partnership between the Library of Congress and the National Endowment for the Humanities.

The website allows users to view, browse, filter, and print recipes published between 1954 and 1960 in the Miami newspaper, Diario las Américas. Recipes are available in Spanish (original language) as well as in English translation. Each recipe page brings together the text of the recipe in English or Spanish, an image of the original recipe in the newspaper, the permalink to the page on which it appears in Chronicling America, and a printable PDF recipe card in either English or Spanish. All recipes are browsable and searchable by title as well as filterable by course, ingredients, and dietary restrictions.

All material used to create this website is available on the Recetas project Github.

Technical Overview

Recetas de las Américas was created using the static site generator Hugo.

Metadata

The text and metadata of each recipe was collected manually by Melissa Jerome, Latin American & Caribbean Digital Initiatives Librarian at the University of Florida and Sarah Tew, Project Coordinator of the US Caribbean & Ethnic Florida Digital Newspaper Project. This spreadsheet is available in Github [https://github.com/SarahTew/recetas/blob/main/source-data/recetas_final.xls].

Translation

Translations into English were first made automatically directly in Excel then edited by Melissa Jerome. Melissa also manually created printable PDF recipe cards for each recipe.

Transformation

The spreadsheet was transformed using Oxygen XML editor by Sarah Tew, into XML, Markdown, and HTML files for use with the static site generator, Hugo.

Hosting

The site is hosted by UF Lib Domains.
